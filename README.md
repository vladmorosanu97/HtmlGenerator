# HTMLgenerator
Transform code from Harramann language to HTML

How to use? 
1. Install k framework

2. Run the command "kompile hrm.k"

If kompile is not recognized then run this : "PATH=$PATH:~/address where you installed it

3. Run "make"

4. Open with your browser "fisier.html"

In start.hrm you can buit your html page with harramann syntax

To find out all the symbols used open Documentatie.pdf
